# Git Abdonor Prompt
Change the colors of your bash, and let the git branch visible.
## Installation

Run the commands:

```bash
mkdir ~/.bash
cd ~/.bash
git clone https://github.com/abdonor/bash-color.git
chmod 777 -R ~/.bash/
~/.bash/bash-color/run.sh
exec bash
```

You are now probably seeing your new bash colours.
